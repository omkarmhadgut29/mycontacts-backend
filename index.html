Node.js API Documentation

User API

GET Endpoints

Endpoint Description Requirements
/api/users/current/${user_id} Get login user. Bearer token


POST Endpoints

Endpoint Description Requirements
/api/users/register Register a new user username, email, password
/api/users/login Login the user and get access Token email, password

Contacts API

GET Endpoints

Endpoint Description Requirements
/api/contacts Get all contact of user Bearer token
/api/contacts/${contact_id} Get perticular contact of user Bearer token


POST Endpoints

Endpoint Description Requirements
/api/contacts Create new contact for user. Bearer token, name, email, phone


PUT Endpoints

Endpoint Description Requirements
/api/contacts/${contact_id} Update perticular contact of user Bearer token, name, email, phone


DELETE Endpoints

Endpoint Description Requirements
/api/contacts/${contact_id} DELETE perticular contact of user Bearer token
